# Treisman App

Reproduction de l'expérience de Treisman sur la théorie de l'intégration des traits. 

Expérience de psychologie cognitive dans laquelle l'utilisateur doit identifier, le plus rapidement possible, une cible visuellement différente parmi un ensemble de formes.
